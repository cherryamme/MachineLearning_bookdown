though there are a lot of good methods to impute missing values.

In this part, I try to introduce imputate by randomforest.

It seems like easier to used and It is a successful methods in Machine Learning area.

We can just use package randomforest or missForest.

randomforest provide two methods:

  1. na.roughfix() Impute Missing Values by median/mode.
  2. rfImpute() Impute missing values in predictor data using proximity from randomForest.

First we use na.roughfix() to impute Fare|Embarked column by Median|Mode.

Then we use rfImpute() to impute Age column by predictoe.

In test dataset,we use function na.roughfix to impute Fare Values. and use missForest for impute missing Age Values easily
